# Sparx-Reader-Copyable-text

This bookmarklet makes Sparx Reader text copyable so that you can paste it along with the questions to any AI and not have to do your Sparx Reader!!!

It works on most browsers apart from Chrome Mobile because Javascript is not allowed.

**How to set up:**

1. Make a bookmark or favourite for this page
2. Name it something that is not suspicious e.g. Sparx Reader
3. Edit this bookmark / favourite and paste the main.js in the URL field.
4. Run the bookmark on the Sparx Reader website.
5. Enjoy copyable sparx reader!! 😁😁
